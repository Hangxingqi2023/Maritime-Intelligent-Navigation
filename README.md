## Spaitlab-USV180平台简介
具备强大的动力和巡航系统，能够满足外场实验需求； 船体较大，能够携带更多样的负载，用于满足不同高校实验室的科研要求。 平台主要用作中型实验和应用场景下的数据采集、船体与控制模型研究、避障与决策算法研究、 自主停泊技术研究。
